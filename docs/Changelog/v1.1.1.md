## v1.1.1 Release Notes

### 🐛 问题修复

本版本主要修复了 PyAppify 打包后应用无法启动及 GUI 无法显示的问题。

### 📦 下载安装

| 文件 | 说明 |
|------|------|
| **DuanjuApp-v1.1.1-Setup.exe** | ✅ 推荐，PyAppify 安装包，支持自动更新 |
| **DuanjuApp-v1.1.1-PyInstaller.zip** | 绿色免安装版，解压即用 |
| **DuanjuApp-v1.1.1-Nuitka.zip** | 极致性能编译版 |

### 🔧 修复

**PyAppify 环境兼容性**
- 修复 PyAppify 打包后无法启动的问题
- 新增 `is_pyappify_env()` 环境检测函数
- 修复 `resource_utils` 模块对 PyAppify 工作目录的支持

**GUI 显示修复**
- 修复 PyAppify 环境下 GUI 窗口无法显示的问题
- 添加 Qt 平台插件路径自动检测和设置
- 优化窗口显示逻辑，确保窗口正确置顶和激活
- SplashDialog 添加 `WindowStaysOnTopHint` 确保初始化对话框可见

**路径处理优化**
- 修复 `log_manager` 日志目录路径问题
- 修复 `config_manager` 配置文件路径问题
- 修复 `cache_manager` 缓存目录路径问题
- 修复 `image_loader` 图片缓存路径问题
- 修复 `favorites_manager` 收藏文件路径问题
- 修复 `history_manager` 历史文件路径问题

**错误处理增强**
- 所有路径模块添加用户目录回退机制
- 添加早期调试日志输出到 `startup_debug.log`
- 改进启动阶段的错误捕获和提示

### 🔨 构建改进

- 发布文件名现在包含版本号，便于区分不同版本
