## v1.1.0 Release Notes

### 🎉 重大更新：项目架构全面重构

本版本对项目的Release版本进行了彻底的模块化重构，并移除了原有的Debug版本。

### ✨ 新功能

**架构重构**
- 将 2800+ 行的单体 main.py 拆分为清晰的模块化结构
- 新增 src/core - 核心数据模型和主题管理
- 新增 src/data - 数据层（API客户端、缓存、配置、收藏/历史管理）
- 新增 src/services - 业务服务层（分类、搜索、视频、下载、统一服务）
- 新增 src/ui - UI层（控件、对话框、界面、窗口）
- 新增 src/utils - 工具类（异步、错误处理、日志、网络监控、重试等）

**多源 API 适配器系统**
- 新增 Provider 适配器架构，支持多数据源切换
- 实现 cenguigui_adapter - 岑鬼鬼短剧 API
- 实现 duanju_search_adapter - 全网短剧搜索 API
- 实现 uuuka_adapter - 即刻短剧 API
- 新增适配器模板，方便扩展新数据源

**完整测试套件**
- 新增 40+ 测试文件，覆盖所有核心模块
- 集成 pytest 测试框架
- 新增 AI 辅助测试工具

**CI/CD 优化**
- 重构 GitHub Actions 工作流
- 支持 PyAppify、PyInstaller、Nuitka 三种打包方式
- 新增依赖缓存和自动化测试
